Проект: Классификатор светимости звёзд

Что сделано:
- начальная база знаний полностью заполнена значениями из научной работы;
- база знаний хранится в data/knowledge_base.json и редактируется через GUI;
- можно добавлять и удалять классы и свойства;
- можно редактировать возможные диапазоны свойств;
- можно редактировать описание свойств каждого класса;
- можно редактировать диапазоны значений свойств для каждого класса;
- реализована проверка полноты знаний;
- реализован решатель по методу опровержения гипотез;
- добавлена ML-модель на sklearn MLPClassifier;
- модель используется только для разрешения неоднозначности между уже подходящими по правилам классами;
- есть отдельный скрипт train_model.py для переобучения модели по текущей базе знаний.

Запуск приложения:
python main.py

Переобучение модели:
python train_model.py

Файлы данных:
- data/knowledge_base.default.json — эталонная база знаний из работы;
- data/knowledge_base.json — рабочая редактируемая база знаний;
- models/star_luminosity_mlp.joblib — обученная ML-модель;
- models/star_luminosity_mlp.meta.json — метаданные модели.

Замечание:
ML-часть адаптирована под твой классификатор светимости звёзд. В качестве примера были взяты присланные файлы с виноградным классификатором, но признаки, классы, данные и логика теперь относятся только к задаче классификации светимости звёзд.
