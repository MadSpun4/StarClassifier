# Классификатор светимости звёзд

Приложение для классификации звёзд по их характеристикам. Проект сочетает экспертную систему на базе правил и ML-модель: основной результат определяется по базе знаний, а модель `sklearn` используется для разрешения неоднозначностей между уже подходящими классами.

## Возможности

- графический интерфейс на `tkinter`;
- редактирование базы знаний через GUI;
- добавление и удаление классов и свойств;
- настройка допустимых диапазонов свойств;
- редактирование описаний свойств и диапазонов значений по классам;
- проверка полноты базы знаний;
- классификация по методу опровержения гипотез;
- переобучение ML-модели по текущей базе знаний.

## Требования

- Python 3.10 или новее;
- `pip`;
- для Windows обычно ничего дополнительно для `tkinter` ставить не нужно, он входит в стандартную установку Python.

## Установка зависимостей

Из корня проекта выполните:

```powershell
pip install -r requirements.txt
```

## Запуск приложения

```powershell
python main.py
```

Также можно использовать батник:

```powershell
run_ui.bat
```

## Переобучение модели

Если вы изменили базу знаний и хотите переобучить модель:

```powershell
python train_model.py
```

Перед обучением проект проверяет корректность и полноту базы знаний. Если в данных есть ошибки, обучение не начнётся, пока они не будут исправлены в редакторе знаний.

## Структура проекта

- `main.py` — точка входа в графическое приложение;
- `train_model.py` — скрипт переобучения модели;
- `requirements.txt` — список внешних зависимостей;
- `data/knowledge_base.default.json` — эталонная база знаний;
- `data/knowledge_base.json` — рабочая редактируемая база знаний;
- `models/star_luminosity_mlp.joblib` — обученная ML-модель;
- `models/star_luminosity_mlp.meta.json` — метаданные модели;
- `star_classifier/` — исходный код приложения.

## Примечание

ML-часть проекта адаптирована под задачу классификации светимости звёзд. Модель не заменяет экспертную систему, а помогает выбрать наиболее вероятный класс, когда по правилам подходят несколько вариантов.
